# Zdjęcia prelegentów

Wrzucaj tutaj zdjęcia prelegentów.

- Pliki są serwowane pod ścieżką `/prelegenci/<nazwa-pliku>`.
- Po dodaniu zdjęcia dopisz prelegenta w `src/data/speakers.ts`
  (`photo: '/prelegenci/...'`, `name`, `talkTitle`, `bio`).
- Zalecane nazwy bez spacji i polskich znaków, np. `piotr-moncarz.jpg`.
